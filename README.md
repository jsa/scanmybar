A simplistic AppEngine app for generating barcode PNGs. Currently supports [Code 128](http://en.wikipedia.org/wiki/Code_128), sets B & C.

(Usage) Example:

![Hello world!](http://scanmybar.appspot.com/code128/Hello%20world!.png)  
[Hello world!](http://scanmybar.appspot.com/code128/Hello%20world!.png)
